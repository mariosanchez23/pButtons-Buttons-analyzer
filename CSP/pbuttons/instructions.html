1) It is recommended to install the latest pButtons from our ftp to ensure to get the best metrics: 

ftp://ftp.intersys.com/pub/performance/
 
Get the pButtons_package.zip and unzip it. You can check the instructions to load, which is basically this:  
 
For example if the 2008.2.xml file was transferred to the mgr directory the command from %SYS namespace would be:
 
%SYS> do $system.OBJ.Load("pbuttonspkg2008.2.xml","ck")
 
 

 2) Change pButtons log output directory – the default output location is the <cache install path>/mgr folder. For example on unix the path to the log directory may look like this:

do setlogdir^pButtons("/somewhere_with_lots_of_space/perflogs/")

Ensure Caché has write permissions for the directory and there is enough disk space available for accumulating the output files.

 
 

3) Schedule pButtons to start at midnight in the Management Portal > System Operation > Task Manager:

Namespace: %SYS
 Task Type: RunLegacyTask
 ExecuteCode: Do run^pButtons("24hours")
 Task Priority: Normal
 User: superuser
 How often: Once daily at 00:00:01
